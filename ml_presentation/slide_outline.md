# 슬라이드 구성표

> 총 슬라이드 예상: 1~5차시 53장 + 6차시 13장 = 66장
> 원칙: 1슬라이드 = 1메시지. 핵심 비주얼 1개 + 키워드 2~3개
> 부록: PCA / 군집화 / AIC·BIC (별도 파일)

---

## 1차시 (13장) — 왜 ML인가

### [도입] 나는 왜 여기 서 있나 (3장)

| 장 | 제목 | 핵심 비주얼 | 원본 |
|---|---|---|---|
| 0 | 야구가 데이터로 보이기 시작했다 | 야구장 이미지 + "왜 이 타자는 이 공에만 약한가?" | 신규 |
| 0-1 | 10년의 여정 | 타임라인: ~10년대 후반 인터넷 서칭 → 20초 R 회귀분석 → GPT 드래그드롭 → 24 ML모델링 → 25 도메인+시각화 설득 → 26 에이전틱 AI | 신규 |
| 0-2 | 도구는 바뀌었고, 목적은 하나였다 | 좌: 시대별 도구 변화 / 우: "결국 설득" 화살표 수렴 | 신규 |

| 3-1 | 상관관계는 가설의 문이다 ⭐ | 좌: 현상→상관관계→가설→인과관계 4단계 흐름도 / 우: 통계학(인과 추구·다중공선성 신경씀) vs ML(상관으로 충분·블랙박스 허용) 비교표 | 신규 |

## 2차시 (13장) — 자유에는 대가가 따른다

| 장 | 제목 | 핵심 비주얼 | 원본 |
|---|---|---|---|
| 1 | 회귀분석이 다중공선성을 신경쓰는 이유 | Y=aX₁+bX₂에서 X₁·X₂가 연관되면 계수 해석 불가 + "계산 거부" 상황 → 1차시 상관/인과 연결 | 슬2+3 압축 |
| 2 | ML의 대답: 결과만 맞으면 | 블랙박스 허용 전환 다이어그램 | 슬4 |
| 3 | 자유의 대가: 과적합 ⭐ | 기출문제 외우기 학생 비유 | 슬6 |
| 4 | 규제화 등장 | 오차 + β 크기 동시 제약 수직 다이어그램 | 슬7 |
| 5 | Ridge: 계수를 억제한다 | 계수가 0에 가까워지되 0이 되지 않음 → 모든 변수 유지 | 슬8 |
| 5-1 | Lasso: 계수를 제거한다 | 계수가 정확히 0이 됨 → 불필요한 변수 탈락 | 슬9 |
| 5-2 | 언제 Ridge, 언제 Lasso? | 변수 많고 대부분 유의미 → Ridge / 변수 선택이 필요 → Lasso 의사결정 카드 | 슬8+9 보강 |
| 6 | 규제화의 진짜 의미 | Bias 희생 → Variance 감소 트레이드오프 곡선 | 슬10 |
| 7 | 분류 문제 맛보기 + Confusion Matrix | 회귀→분류 전환 + TP/FP/FN/TN 2×2 표 + 정확도 함정 + K-Fold 5분할 | 슬11~12-1·신규 통합 |
| 8 | Train / Validation / Test: 왜 세 개로 나누나 | 삼분할 다이어그램 + "Test는 딱 한 번만" 경고 | 신규 |
| 9 | 어떤 지표로 모델을 평가하나 | 상황별 지표 선택 의사결정표 (Recall vs Precision vs AUC) | 신규 |
| 10 | 실무 함정: Data Leakage | 타임라인 그래프 + 미래 데이터 침투 경고 박스 | 신규 |
| 11 | 예고: 직선으로 안 되는 세상 | 다음 차시 연결 | 슬12-2 |
