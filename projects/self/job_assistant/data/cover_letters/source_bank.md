## 1. POSCO AI 빅데이터 아카데미

**시기:** 2025년 2월 ~
**형태:** 교육 + 팀 프로젝트 2건

---

### 1-A. 철강 불량 예측 — 데이터 분석 프로젝트

**무슨 일이었나:**
- 팀원 대부분이 제조업 관심 → 주제: 철강 연주 공정 불량 예측
- 본인은 철강 도메인 완전 미지 → **공정 원리 먼저 학습** (물리적 메커니즘 → 불량 발생 이론 → 논문 탐색)
- 분석 플로우 설계 → 팀에 방향 제시
- 단순 예측 모델이 아닌 **"실제 도출 가능한 개선안"** 으로 문제 재정의

**어필 포인트:**
| 역량 | 근거 |
|------|------|
| 도메인 러닝 속도 | 모르는 분야를 원리부터 빠르게 파악해 분석 구조화 |
| 문제 정의력 | 데이터 분석 → 비즈니스 의사결정 연결 |
| 팀 내 설계 주도 | 분석 방향 제시, 변경 리스크 고려 |

**구체적 수치/결과:** 분석 플로우 설계 → 팀 방향성 확정, 변경 리스크를 고려한 개선안 제시

**키워드:** 제조업, 도메인 학습, 문제 정의, 비즈니스 임팩트, 팀 기여

---

### 1-B. AI 자율 프로젝트 — 실현가능성 우선 전략

**무슨 일이었나:**
- 주제 선정에서 팀이 막힘 → 본인: **아이디어보다 실현가능성에 가중치**
- AI 환각 문제 인식 → 주어진 시간 내 완수 가능한 범위로 스코프 제한
- **말로 설득하지 않고** WSL에서 프로토타입 직접 구현 → 작동하는 결과물로 설득
- "이론적으로 가능하다"가 아닌 "실제로 돌아간다"

**어필 포인트:**
| 역량 | 근거 |
|------|------|
| 실행력 | 프로토타입 먼저 만들어 팀 설득 |
| 리스크 판단 | AI 환각 문제 사전 인식, 스코프 조정 |
| 커뮤니케이션 | 코드로 증명 — 가장 빠른 설득 방법 |

**키워드:** 프로토타입, 실행력, 리스크 관리, 팀 리드

## 2. 논문 — 연준 통화정책 기대 분석

**시기:** 2025년 여름 이후
**형태:** 개인 연구 (AI와 협업)
**주제:** 2024년 하반기 연준 통화정책 전환기 — 시장의 정책금리 기대를 설명하는 거시경제 변수 식별

---

### 핵심 내용

**문제:**
- 시장은 빠르게 움직였지만 설명은 사후적·정성적
- 50개 거시경제 변수 → 과적합 vs 정보 손실 트레이드오프

**설계한 3단계 추정 프레임워크:**
1. LASSO → 고차원 변수 중 설명력 있는 변수 선택
2. Post-LASSO OLS → LASSO 선택 변수로 재추정, 축소 편향 제거
3. Newey-West HAC 표준오차 → 자기상관·이분산 보정

**롤링 검증:**
- 초기 결과: Baa 스프레드가 핵심 변수처럼 보임 → **의심함**
- 100거래일 윈도우, 20거래일 이동, **68회 반복 추정**
- 결과: Baa 스프레드는 선택 빈도 불안정, 계수 부호 역전
- Ret_Corp_InvGrade: **73% 선택 빈도, 100% 부호 일관성** → 가장 강건한 변수

**구조 변화 검증:**
- 2024년 12월 Dot Plot 전후 Chow Test
- R² 0.67 → 0.76 상승
- 달러 인덱스·신용 스프레드 계수가 구조적으로 변화
- "변수 값이 바뀐 것이 아니라 시장이 정보를 처리하는 방식 자체가 변함"

**어필 포인트:**
| 역량 | 근거 |
|------|------|
| 검증 집착 | 좋아 보이는 초기 결과 의심 → 68회 반복 검증 |
| 통계 설계력 | LASSO + Post-LASSO + HAC 3단계 체계 설계 |
| 인사이트 도출 | 구조 변화 발견 — 시장 반응함수 자체가 바뀜 |
| 분석 태도 | 예측보다 설명력, 안정성 검증 우선 |

**수치 증거:**
- 50개 변수 → LASSO 선택
- 68회 롤링 추정
- R² 0.67 → 0.76
- Ret_Corp_InvGrade: 선택 빈도 73%, 부호 일관성 100%

**키워드:** LASSO, Post-LASSO OLS, Newey-West HAC, 롤링 분석, Chow Test, 구조 변화, 거시경제 변수, 연준

**관련 파일:**
- `~/projects/forecast/` — 연장선상의 Fed 정책 예측 파이프라인
- `~/projects/forecast/forecasting_20251218.py` (156KB)

## 5. AI 설계 프로젝트 (로컬 Git)

---

### 5-A. Conditional Density Estimation — 시장 리스크 모델링 (`~/market_density_project/`)

**핵심 아이디어:**
- 점 추정(포인트 예측) 대신 **확률 밀도 함수(PDF) 예측**
- 시장 피처 조건부 분포 추정 → 꼬리 리스크 포착

**5단계 파이프라인:**
1. Gaussian Baseline
2. Heteroscedastic Model
3. Gaussian Mixture Model (GMM)
4. Neural Mixture Density Network (MDN)
5. 자동 단계 전환 (통계 검정 기반)

**구현 내용:**
- 피처 엔지니어링: 실현 변동성, 거래량 Z-Score, RSI
- 리스크 엔진: VaR, Expected Shortfall, 유럽형 옵션 가격
- 전략: 변동성 타겟팅, 꼬리 헤지
- 자동화 검정: Breusch-Pagan, Kurtosis, Hartigan Dip
- 평가 지표: NLL, CRPS, PIT scoring
- Streamlit 대시보드
- Optuna 하이퍼파라미터 최적화

**어필 포인트:**
| 역량 | 근거 |
|------|------|
| 방법론 설계 | 단계별 모델 복잡도 점진 상승, 자동 전환 |
| 금융 도메인 | VaR, ES, 옵션 가격, 꼬리 리스크 |
| 엔지니어링 | 테스트, 대시보드, 하이퍼파라미터 최적화 |

**관련 파일:** `~/market_density_project/pipeline.py`, `stages.py`, `backtest_engine.py`
