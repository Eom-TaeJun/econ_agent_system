## 5. 웨이브릿지 상세 분석

### 무슨 회사인가
- 기관/법인 대상 디지털자산 프라임 브로커리지
- 구조화 상품, 체계적 거래 전략, 퀀트 리서치
- 직원 55명, 평균연봉 5,932만원

### Junior Quantitative Researcher 요건
- 금융·수학·통계·물리 등 정량 분야 학위
- 0~3년 실제 데이터 기반 심화 연구 경험
- Python + Unix 환경
- 영어 비즈니스 커뮤니케이션 필수

### 내 소스 매핑
| 요건 | 대응 소스 |
|------|---------|
| 정량 분야 학위 | 과학고 + 경제학 |
| 데이터 기반 심화 연구 | 논문 (LASSO + 68회 롤링 + Chow Test) |
| 가격 변동 통계 모델 | market_density (CDE/VaR/ES/MDN) |
| 체계적 거래 전략 | regime_detection + EIMAS 백테스팅 |
| Python | 전 프로젝트 공통 |
