## 다날이 진짜 원하는 역량 5가지

### 1. 투자 리서치 + IM 작성 능력 ★★★
IM(Investment Memorandum) 구조:
```
1. Executive Summary    — 투자 논리 요약
2. Company Overview     — 사업 모델, 제품
3. Market Opportunity   — TAM, 경쟁 구조
4. Business Model       — 수익 구조, 핵심 KPI
5. Investment Thesis    — 왜 지금, 왜 이 회사
6. Risks               — 리스크 + 대응 논리
```
인턴 담당 파트: 2(Company Overview), 3(Market), 6(Risks)

### 2. 거시경제 + 금융 지표 해석 능력 ★★★
다날이 보는 주요 지표:
- 금리: Fed Funds Rate, 한국 기준금리
- 환율: USD/KRW
- 크립토: BTC 도미넌스, 스테이블코인 유통량
- 온체인: TVL, DEX 거래량

### 3. 글로벌 핀테크/디지털자산 도메인 지식 ★★★
다날이 관심 갖는 영역:
- 스테이블코인 시장 (Tether, USDC, 원화 스테이블코인)
- EU MiCA 규제 동향
- 글로벌 결제 핀테크 (Stripe, Adyen, Wise)
- AI 에이전트 결제 (x402 프로토콜 계열)

### 4. 데이터 시각화 + 리포트 작성 능력 ★★
원하는 산출물 형태:
- 깔끔한 차트 (matplotlib, plotly)
- 구조화된 Markdown 또는 PDF 리포트
- 한 페이지 Executive Summary

### 5. 영문 리서치 독해 + 요약 능력 ★★
영문 리포트 → 한국어 요약 리포트 변환

- 산출물: IM 구조 (Overview / Market / Risks) Markdown 출력
- 다날 연결: "IM 초안 작성 보조" JD와 1:1 매핑

### Option C. 글로벌 핀테크/디지털자산 주간 리서치 브리핑
- 데이터: RSS 피드 + 거시경제 지표 + 온체인 데이터

- 다날 연결: "시장 동향 모니터링 및 리서치 요약 자료 작성" JD와 직결
