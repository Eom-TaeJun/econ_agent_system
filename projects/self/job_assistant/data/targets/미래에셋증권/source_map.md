## 전략 방향

### 미래에셋 특징
- 글로벌 IB 지향, 해외 비즈니스 비중 높음
- 현직자 후기: "인턴에게 기대치 낮음 → 자기주도 학습 의지 강조 시 차별화"
- 자유 형식 이력서 → 구조 자체가 경쟁력

### 섹터 특화 (운송·유틸리티·지주)

| 섹터 | 핵심 KPI | 자소서 활용 방향 |
|------|---------|----------------|
| 운송 (항공) | RPK, ASK, 탑승률, 유가, 환율 | 유가·환율 민감도 분석 경험 연결 |
| 운송 (해운) | BDI, SCFI, 선복량 | - |
| 유틸리티 | 전력판매단가, 연료비, RAB | 정부 규제·수익성 관계 이해 |
| 지주 | NAV 할인율, 지주사 디스카운트 | NAV 방법론 언급 |

## 보유 소재 진단

| 소재 | 활용 가능? | 프레이밍 방향 |
|------|-----------|--------------|
| Fed 논문 (LASSO, Chow Test) | ✅ | 거시 분석 → 섹터 영향 이해 |
| eco_system_v2 (Python + FRED) | ✅ | 데이터 업데이트 역량 |
| 영어 스피킹 자격증 | ✅ | 영문 리포트 독해 (외국어 우대) |
| POSCO ML 경험 | ✅ | 모르는 도메인 빠르게 파악 |
| eimas | ❌ | 쓰지 않음 |

## 이력서 구조 (자유 형식)

**추천 구성:**
1. 핵심 역량 요약 (3줄)
2. 지원 동기 (운송·유틸·지주 섹터 + 미래에셋)
3. 관련 경험
   - Fed 논문: 거시 분석 역량
   - eco_system_v2: Python 데이터 처리
   - POSCO ML: 도메인 학습 속도
4. 역량
   - Python, MS Office
   - 영어 (스피킹 자격증)
