# 05. AI 컨설턴트 세팅

## 목적

특정 도메인 자료(산업 보고서, 규제 문서, 내부 자료)를 NotebookLM에 쌓아두고
도메인 전문가 수준의 컨설팅 질문에 답변받는 환경을 구성한다.

---

## notebooklm.google.com에서 수동으로 할 것

1. 새 노트북 생성 (예: "컨설턴트_[도메인]")
2. 소스 추가 (도메인 특화 자료):
   - 산업 리포트 PDF (McKinsey, Deloitte, 증권사 리포트 등)
   - 규제/법령 문서 URL
   - 내부 전략 문서 업로드
   - 경쟁사 분석 자료
3. 노트북 URL 복사

---

## Claude Code에서 사용하는 MCP 명령어

```
# 노트북 등록
"Add https://notebooklm.google.com/notebook/xxx to library with tag consultant"

# 노트북 선택
"Select notebook with tag consultant"

# 컨설팅 질문 예시
"Ask: 이 산업의 향후 3년 핵심 트렌드 5가지를 전문가 관점에서 설명해줘"
"Ask: 신규 진입자가 직면하는 주요 장벽은 무엇이고, 극복 전략은?"
"Ask: 규제 변화가 비즈니스 모델에 미치는 영향을 분석해줘"
"Ask: 이 시장에서 성공한 사례와 실패한 사례의 차이점은?"
"Ask: [특정 전략]을 실행할 때 예상되는 리스크와 완화 방안은?"
```

---

## 예상 출력 형태

- 트렌드 5가지 (각 1~2문단 설명)
- 진입 장벽 매트릭스
- 규제 영향 분석표
- 성공/실패 사례 비교 표
- 리스크 완화 액션플랜
