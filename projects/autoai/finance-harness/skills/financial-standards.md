# 금융 분석 리포트 표준

> Finance Harness 에이전트가 투자 리포트 작성 시 준수해야 할 형식·품질 기준.
> FinRobot Expert_Investor 패턴 기반.

---

## 1. 리포트 구조 (필수)

### 섹션 순서
1. **Executive Summary** (3줄 이내)
   - 핵심 투자 판단 한 줄
   - 주요 근거 한 줄
   - 리스크 요약 한 줄

2. **핵심 재무 지표** (테이블 형식 필수)

| 지표 | 현재값 | 업종 평균 | 5년 평균 | 평가 |
|------|--------|----------|---------|------|
| P/E | | | | |
| EV/EBITDA | | | | |
| PBR | | | | |
| ROE | | | | |
| 부채비율 | | | | |

3. **밸류에이션 분석**
   - DCF 내재가치 (할인율 근거 명시)
   - 피어 비교 멀티플 (최소 3개 비교 기업)
   - 히스토리컬 멀티플 (5년 분위수 위치)

4. **기술적 분석**
   - RSI (14일): 과매수>70, 과매도<30
   - MACD (12/26/9): 신호선 교차 방향
   - 52주 고/저 대비 현재가 위치

5. **리스크 요인**
   - 상승 촉매 (Bull case) 3개
   - 하락 리스크 (Bear case) 3개
   - 리스크 등급 (risk-mgr 결과)

6. **투자 의견**
   - Buy / Hold / Sell
   - 목표가 (12개월)
   - 근거 2-3줄

---

## 2. 인용 규칙

### 필수 출처 표기
- 재무 데이터: "(출처: FinnHub, {날짜})"
- 뉴스/공시: "(출처: Perplexity, {URL})"
- 거시지표: "(출처: FRED, {Series ID})"
- 애널리스트 컨센서스: "(출처: {기관명}, {날짜})"

### 금지 표현
- "업계 전문가에 따르면..." (출처 없이)
- "일반적으로 알려진..." (검증 불가)
- "~할 것으로 예상된다" (근거 없이)

---

## 3. 숫자 표기 기준

- 퍼센트: 소수점 1자리 (e.g., 12.3%)
- 주가: 소수점 2자리 (e.g., $182.45)
- 시가총액: 억/조 단위 (e.g., 3.2조달러)
- 비율: 소수점 2자리 (e.g., P/E 28.45x)

---

## 4. 면책 문구 (필수)

모든 리포트 하단에 포함:
> **투자 면책 고지:** 본 분석은 정보 제공 목적이며 투자 권유가 아닙니다.
> 과거 수익률은 미래를 보장하지 않습니다.
> 실제 투자 결정 전 전문 금융 자문사와 상담하세요.
