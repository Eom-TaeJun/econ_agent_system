# 암호화폐·온체인 분석 프레임워크

## GENIUS Act — 미국 스테이블코인 규제 기준

### 지불용 스테이블코인 5대 요건

```
1. 1:1 달러 연동
   준비금: USD, 미국 단기국채, 단기 연방채 등 고유동성 저위험 자산만 허용

2. 허가 발행자 (Permitted Issuer)
   연방 및 주 감독기관의 정식 승인 필수 (FINCEN, NYDFS 등)

3. 투명성 및 감사
   정기 감사, 공개 보고, AML/CFT 준수 의무

4. 수익 지급 금지
   staking / yield-bearing 구조 → 지불용 분류에서 제외 (투자상품으로 재분류)

5. 결제 기능 중심
   투자수단이 아닌 지불/송금 목적으로만 발행/유통
```

### 주요 스테이블코인 GENIUS Act 준거 비교

| 코인 | 발행사 | 준비금 구조 | GENIUS 준거도 | 핵심 이슈 |
|------|--------|------------|--------------|----------|
| **USDC** | Circle (미국) | 단기국채 ~80% + 현금 ~20% | **높음** | 월간 PwC 감사, FINCEN+NYDFS 허가 |
| **USDT** | Tether (홍콩) | 혼합 (과거: 회사채·대출·암호화폐 포함) | 부분 | 준비금 리스크, 감사 투명성 논란, 해외 발행 |
| **DAI** | MakerDAO (탈중앙) | 암호화폐 담보형, 초과담보 | 불충족 | 허가 발행사 없음, 암호화폐 준비금 |
| **USDe** | Ethena Labs (싱가포르) | ETH+stETH+파생상품 델타 헤지 | 불충족 | yield-bearing 구조 (수익형 → 투자상품) |
| **WLFI** | Trump Family | 토큰 판매 기반 | 불충족 | 양도 불가, 내부자 ~70% 보유, 유동성 없음 |

**시장 현황 (2025 기준):**
- USDT: 시장점유율 ~67%, 시가총액 ~$990B
- USD 스테이블코인 총 시장: ~$2,250억 (전 세계 스테이블코인의 99%)

## 암호화폐 화폐론적 분류 (M-집계 매핑)

| 암호자산 | M-집계 유사체 | 근거 |
|---------|-------------|------|
| **BTC** | Base Money / M0 유사 | 고정 공급(2,100만), 중앙은행 통제 외부, 높은 변동성, "디지털 금" |
| **스테이블코인** | M1 → M2/M3 가교 | 1:1 달러 페그, 가장 유동적인 결제 수단, 준비금(현금+국채) = MMF와 유사 |
| **ETH** | 금융 인프라 (화폐 아님) | 가스비용 유틸리티 토큰, dApp/DeFi/NFT 플랫폼, 공급 제한 없음 |

**BTC vs ETH 구조 비교:**

| 특성 | Bitcoin | Ethereum |
|------|---------|----------|
| 역할 | 디지털 금 / 가치 저장 | 프로그래밍 가능한 블록체인 인프라 |
| 공급 | 21M BTC 고정 | 무제한 |
| 기술 | 단순 전송 트랜잭션 | 스마트 컨트랙트 |
| 합의 | PoW (채굴) | PoS (검증자 스테이킹) |

**Bitcoin Funding Rate = 사실상 BTC 기준금리:**
```
무기한 선물 펀딩 레이트가 양(+)이면: 롱 포지션이 많음 = 과열 신호
펀딩 레이트가 음(-)이면: 숏 지배 = 약세 또는 헤지 수요
→ Bitcoin이 가치저장에서 금융등급 자산으로 진화한 증거
```

## BTCFi — Bitcoin 금융 생태계 진화

### 5단계 진화 프레임워크

```
Stage 1: Bitcoin (가치 저장)
Stage 2: BTCFi (금융 자산화 — 대출, 수익, 담보)
Stage 3: Layer 2 (기술 인프라 — 속도·비용 해결)
Stage 4: 온체인 금융 시장 형성
Stage 5: 디지털 금융 경제 완성
```

### BTCFi 구현 방식

```
1. 래핑 (Wrapping)
   BTC → 다른 체인에서 사용 가능한 토큰화
   예: wBTC (Ethereum), tBTC

2. 사이드체인 (Sidechain)
   Bitcoin 옆에 독립 체인 + 브릿지
   예: Stacks (PoX), RSK/Rootstock (RBTC), Mintlayer

3. Layer 2
   Bitcoin 메인체인 직상위 확장 레이어
   예: Merlin Chain (ZK-Rollup), BEVM, Lightning Network
```

### L2 vs 사이드체인 결정 매트릭스

| 속성 | 사이드체인 | Layer 2 |
|------|-----------|---------|
| 구조 | 메인체인 옆 독립 체인 | 메인체인 위 확장 레이어 |
| 보안 | 자체 합의 메커니즘 | 메인체인 보안 상속 |
| 연결 | 2-way 페그 (브릿지 필요) | Rollup/채널 직접 연결 |
| 예시 | Stacks, RSK, Mintlayer | Merlin Chain, BEVM |
| 선택 기준 | 독립성·철학적 탈중앙화 우선 | 보안·신뢰성 우선 |

## 스테이블코인 거시경제 임팩트

### 미국 T-bill과의 연결

```
스테이블코인 발행사 = 미국 단기국채의 구조적 수요자

$2,250억 시장 × ~80% 단기국채 배분
→ 약 $1,800억의 T-bill 구조적 수요 창출

함의:
  스테이블코인 시장 성장 = T-bill 수요 증가 = 미국 단기 조달 비용 하락
  스테이블코인 대규모 상환 = 급격한 T-bill 매도 압력
```

### 미국 CBDC 포기 → 민간 스테이블코인 채택

```
Anti-CBDC Surveillance State Act (미국):
  연방준비제도의 소매 CBDC 발행/테스트 명시적 금지
  이유: 금융 프라이버시 우려 + 민간은행 중개 역할 보존

결과:
  미국의 디지털 달러 전략 = 규제된 민간 스테이블코인 (GENIUS Act 준거)
  USDC가 사실상 '제도권 디지털 달러' 포지션
```

### 신흥국 달러화 리스크 (한국 분석)

```
달러화 촉진 경로:
  스테이블코인 → 자국 통화 대체 → 통화정책 중성화

한국 완충 요인:
  ✅ 대규모 외환보유고
  ✅ 성숙한 FX 파생상품·헤징 시장
  ✅ 강력한 자본 감독 체계
  ✅ 높은 중앙은행 신뢰도 (낮은 인플레 기대)

한국 취약 요인:
  ⚠️ 성장하는 암호화폐·디지털 금융 인프라
  ⚠️ 전통 은행 시스템 우회 가능한 신규 자본 흐름
```
