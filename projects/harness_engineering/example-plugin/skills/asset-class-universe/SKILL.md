# 자산 클래스 유니버스 프레임워크

## 전통 자산 클래스

```
Financial Assets (가장 중요한 wealth 소스):

1. Equity (주식)
   국내: KOSPI, KOSDAQ
   해외: S&P500, NASDAQ
   ETF:  섹터별, 테마별 (분석 진입점)

2. Bond (채권)
   정부채(Government), 회사채(Private)
   단기채 → 파킹용 = 시장 불안 신호
   레버리지 가능: 현금의 15%만 있으면 거래 가능

3. FX (외환)
   정부 개입 변수 → 예측 가장 어려움
   파생상품 포함

4. Commodity (원자재)
   금, 백금, 희토류
   달러 불신 시 대안 자산
```

## 확장 자산 클래스 (현재 진행형)

```
5. Cryptocurrency
   BTC: 자산 전환 진행 중 (기관 채택)
   ETH: 스마트컨트랙트 플랫폼
   Altcoin: 투기적 성격

6. Stablecoin
   달러 불신 시대의 통화 역할
   USDT, USDC → 신흥국 금융 접근성 해결 가능성
   GENIUS Act 등 규제 명확화 후 급성장 예상

7. Real World Asset (RWA) - 가장 큰 성장 가능성
   부동산 (분기별 완만한 변동, 장기 수익 높음)
   광산/자원 채굴권
   예술품 → 50만원으로 살 수 있는 세상 (토큰화)
   특징: 유동성 낮음 but 접근성 혁명적 개선 중

8. 신규 ETF 영역 (곧 등장 예상)
   블록체인 ETF
   희토류/자원 ETF
   드론/방산 ETF
   양자컴퓨터 ETF
```

## 자산별 특성 비교

| 자산 | 리턴 | 리스크 | 유동성 | 핵심 특이사항 |
|------|------|--------|--------|--------------|
| 주식 | 높음 | 높음 | 높음 | 인덱스 영향 70-80% |
| 채권 | 낮음 | 낮음 | 중간 | 단기채=시장 불안 신호 |
| FX | 중간 | 중간 | 높음 | 정부 개입 변수 |
| 금 | 중간 | 낮음 | 높음 | 달러 위험 헷지 |
| 암호화폐 | 매우 높음 | 매우 높음 | 높음 | 규제 변수 |
| 부동산 | 높음 | 중간 | 낮음 | 장기 유리 |
| RWA | 불확실 | 높음 | 낮음 | 토큰화로 접근성 증가 |

## 시장 상황별 자산 선택

### 금리 기준 (미국 10년물)
```
< 4.0%    : 주식 우호적, 성장주 선호
4.0~4.7% : 경계 구간, 퀄리티 팩터 선호
> 4.7%   : 방어 자산 비중 증가, 단기채 파킹
```

### Risk-On / Risk-Off
```
Risk-On 환경:
  성장주, 기술주, 반도체 ETF 주목
  EM 자산, 고수익채 선호

Risk-Off 환경:
  단기채 거래 급증 → 파킹 수요 확인
  금 상승 → 달러 위험 신호
  모든 ETF 동반 하락 → 시스템 위기 (2007-2009 패턴)
  현금/단기국채 최우선
```

### 현재(2025-2026) 주목 자산
- **반도체**: 미국의 중국 견제 수혜 → 하이닉스, 삼성전자
- **방산/드론**: 지정학적 긴장 지속
- **스테이블코인**: GENIUS Act 이후 급성장 예상
- **금**: 달러 불신 지속 시 유리
- **희토류 관련**: 미중 갈등 장기화 수혜

## 🔴 2026년 자산 환경 업데이트

### 스테이블코인·크립토 규제 실행 단계
```
GENIUS Act (미국, 2025년 7월 통과):
  - 구현 규정 발표: 2026년 7월 18일
  - 규정 발효: 2027년 1월 18일
  - 핵심: 1:1 고품질 유동자산 준비금 의무화

글로벌 동조화:
  캐나다: GENIUS Act 미러링 초안 (2025.11)
  영국·홍콩·일본: 기존 제도 강화
  → 2026 = 스테이블코인 규제 이론→실행 전환의 해

투자 임플리케이션:
  규제 명확화 → 기관 진입 장벽 낮아짐
  수혜: 컴플라이언트 스테이블코인 발행사 (USDC 등)
  관련주: COIN, 결제 인프라, 커스터디 서비스
```

### RWA (Real World Asset) 토큰화 — 가속화
```
현황 (2026년 초):
  온체인 RWA 공급: $36B+ 돌파
  BlackRock BUIDL 펀드: $500M+ 규모
  TradFi × DeFi 융합 = 현재 진행형 현실

성장 중인 카테고리:
  1. 토큰화 국채/머니마켓 펀드 (가장 큰 비중)
  2. 프라이빗 크레딧 온체인화
  3. 부동산 토큰화 (유동성 혁신)
  4. 원자재·채굴권 (희토류, 금 채굴권)

핵심 플레이어: ONDO, BlackRock, Franklin Templeton
```

### 2026 에셋 성과 환경
```
EM > DM (초반 성과):
  이유: 달러 약세 기대 + 무역 다변화 + 강한 내수
  주목 지역: 인도, 동남아 (관세 회피 생산기지)

관세 직격 vs 수혜 구분:
  피해: EU 자동차·럭셔리, 중국 직수출 기업
  수혜: 미국 국내 제조, 인도·베트남 대안 공급망
       반도체·AI 인프라 (전략적 보호)

방어 팩터 선호:
  Low Volatility, Quality, Dividend
  → 관세 불확실성이 해소될 때까지 유효
```

## 섹터 로테이션 원칙

```
Active > Passive 전환 시대:
  Index ETF는 포화 → Active ETF(인덱스 외 알파 발굴)로 이동
  ETF 내 보이지 않는 알파 발굴이 핵심

분석 순서:
  1. 거시 레짐 판단 (Growth-Inflation Matrix)
  2. 레짐에 맞는 섹터 방향 설정
  3. ETF 비율로 빅머니 흐름 확인
  4. 개별 종목 알파 발굴
```
