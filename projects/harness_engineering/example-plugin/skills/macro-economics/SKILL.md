# 거시경제 분석 표준

## 핵심 인식론

**경제학의 근본 질문은 항상 인과관계(causality)다.**

- "왜 올라가나?" (단순 상승 확인 X)
- 상관관계는 있지만 인과관계는 블랙박스 → ML의 explainability가 이를 해결하려는 시도
- 정책 실패 분석, 수익률 기대 이하 원인 → 인과관계로만 설명 가능

```
소득 증가 → 소비 증가 → 경제성장  (이상적 인과)
But 현실: 상관관계는 있으나 방향/강도 불확실
```

---

## GDP 분해 구조

```
Y = C + I + G + NX

C = f(wealth, 소득)
  → wealth의 핵심: financial asset (equity, bond, FX파생, commodity)
  → 코로나 이후 M2 팽창이 자산시장 직접 반영

M/P = L(R, Y)
```

**역사적 전환점:**
- **1971년**: 변동환율제 도입 → 현대 금융시장의 시작점 (달러 패권 재편 가능성의 원점)
- **2007년 이후**: 포트폴리오 이론이 시장에 반영됨
- **코로나 이후**: M2 = 시장 핵심 변수

## 달러 대안 자산 계층

```
금 상승 = 세상이 위험하다는 신호 = 달러에 대한 위험 신호
달러 불신 → 대안 신용 보증 수단으로 이동
```

**자산 계층 순서:**
1. 금, 백금 (전통적 안전자산)
2. 암호화폐 (새로운 신용 보증 수단 논의)
3. 스테이블코인 (통화의 역할 대체 가능성)
4. RWA (부동산, 광산 채굴권, 예술품 토큰화)

**중장기 구도:** 금융으로 견제하는 미국 vs 희토류로 견제하는 중국

## 미국 부채 임계값

```
4.0% 미만  : 정상, 주식 우호적
4.0~4.7%  : 경계 구간
4.7% 초과  : 채무불이행/정부셧다운 위험 → 방어 자산 비중 증가
```

## 🔴 2026년 현재 환경 업데이트

### Fed 현황 (2026-01 기준)
- **기준금리**: 동결 유지. 시장은 연내 2회 인하(50bp) 전망 → 최종금리 3.0~3.25%
- **Fed 의장 교체**: Kevin Warsh, Powell 임기 만료(2026년 5월) 후 취임 예정
  → 의장 교체 불확실성 = 추가 정책 변수
- **인플레이션**: 관세 효과 소멸되면서 Core PCE 2% 재접근 예상
- **경기침체 확률**: 약 30~35% (6개월 전 40%에서 하락)

### 관세(Tariff) — 새로운 핵심 거시 변수
```
미국 실효관세율이 100년래 최고 수준으로 급등 (2025년~)

주요 영향:
  단기: 관세 효과로 물가 상방 압력 (Fed 인하 제약)
  중기: 공급망 재편 비용 → 기업이익 압박
  장기: 글로벌 교역 구조 재편

섹터별 차별화:
  직격: 제약(200% 관세 논의 중), 자동차, 럭셔리
  수혜: 반도체/AI(전략적 예외), 미국 국내 제조업
  EM: 초반 달러 약세 + 무역 다변화 수혜로 선진국 아웃퍼폼

포트폴리오 대응:
  → 저변동성(Low-Vol), 퀄리티 팩터 선호
  → 인도·동남아 등 대안 공급망 수혜 지역 주목
  → 공급망 익스포저 확인 필수 (매출의 對中/對美 비중)
```

### 미국 성장 전망
- 2026 GDP: +2.2% 예상 (재정·통화 완화 효과)
- 노동시장: 완만한 냉각세 지속, 임금 상승률 둔화 중
- 소비: 여전히 견조하나 관세 효과로 불확실성 증가

## 핵심 지표 체계

### 성장 (Growth)
- **GDP**: 명목/실질 구분 필수. QoQ SAAR vs YoY 맥락 명시
- **산업생산(IP)**: 제조업·유틸리티·광업 분리 해석
- **소매판매**: 자동차·가솔린 제외 Core 지표 우선

### 물가 (Inflation)
- **CPI vs PCE**: Fed 공식 목표는 Core PCE (2%)
- **Breakeven**: 시장 기대 인플레이션 = TIPS 스프레드
- **PPI → CPI 선행**: 생산자물가로 소비자물가 방향 예측

### 고용 (Labor)
- **비농업고용(NFP)**: 월간 ±50k는 잡음, 3개월 이동평균 우선
- **실업률 vs U-6**: Headline 실업률 + 광의 실업률(U-6) 함께 해석
- **시간당 임금 상승률**: 임금 인플레이션 → 서비스 CPI 선행 지표

### 금리 (Rates)
- **수익률 곡선**: 10Y-2Y 스프레드 역전 = 경기침체 선행 시그널 (평균 12-18개월 선행)
- **실질금리 = 명목금리 - Breakeven**: 금융여건 긴축 여부의 핵심
- **Fed Funds Rate**: 점도표(Dot Plot)와 시장 기대(Fed Funds Futures) 비교

### 신용 (Credit)
- **HY OAS (High Yield OAS)**: 리스크 온/오프 바로미터. >500bp = 위험 구간
- **IG OAS (Investment Grade OAS)**: 시스템 리스크 확대 여부
- **SOFR-OIS 스프레드**: 은행 간 신용 리스크 (구 TED Spread)

### 유동성 (Liquidity)
- **Fed 대차대조표**: QT/QE 방향 = 시장 유동성 직결
- **RRP (역레포)**: 과잉 유동성 지표. 감소 = 유동성 흡수 시작
- **TGA (재무부 현금)**: TGA 축소 → 시중 유동성 공급 효과

## 분석 절차

1. **데이터 검증**: 시계열 연속성, 계절조정(SA) 여부, 기준연도 확인
2. **추세 분리**: HP 필터 또는 이동평균으로 사이클 vs 추세 분리
3. **상관관계 분석**: 상관관계 ≠ 인과관계. Granger Causality 또는 경제 이론으로 방향성 근거 제시
4. **레짐 판단**: 성장·인플레이션 2x2 매트릭스로 4가지 레짐 분류
5. **불확실성 표시**: 신뢰구간(95% CI) 또는 시나리오 범위 항상 병기

## 거시 레짐 매트릭스

| | 인플레이션 ↑ | 인플레이션 ↓ |
|---|---|---|
| **성장 ↑** | Overheating (금리 인상 압력) | Goldilocks (최적 환경) |
| **성장 ↓** | Stagflation (최악) | Recession (침체/완화 전환) |

## 출력 규칙

- 수치는 항상 단위(%, bp, $B 등)와 기간 명시
- "긍정적/부정적" 대신 방향과 크기로 서술 ("전월 대비 +2.3%, 2년래 최고")
- 정책적 함의는 항상 근거 지표와 함께 제시
- 불확실성이 높을 때는 "현재 데이터로는 판단 불가" 명시
