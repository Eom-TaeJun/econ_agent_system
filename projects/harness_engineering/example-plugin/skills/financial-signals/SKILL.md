# 금융 시그널 해석 표준

## 변동성 & 공포 지수

### VIX 해석
| 구간 | 의미 | 대응 |
|------|------|------|
| < 15 | 과도한 낙관 / 컴플레이선시 | 리스크 관리 강화 |
| 15–20 | 정상 구간 | 중립 |
| 20–30 | 불안 구간 | 헤지 비용 상승 |
| > 30 | 공포 구간 | 역발상 매수 검토 (단, 방향 확인 필수) |
| > 40 | 패닉 / 시스템 위기 | 현금 비중 극대화 후 저점 탐색 |

### VIX 구조 분석 (Bekaert et al. 방법론)
- **VIX = Risk Premium + Uncertainty**: 단순 변동성이 아닌 두 요소 분리
- VIX spike 시 → 어느 요소 주도인지 판단이 대응 전략 결정

## 수익률 곡선

### 핵심 스프레드
- **10Y - 2Y**: 경기침체 선행 (역전 후 평균 12-18개월)
- **10Y - 3M**: Fed가 중시하는 스프레드
- **30Y - 5Y**: 장기 성장 기대 반영

### 해석 원칙
- 역전 = 침체 예측이 아닌 침체 확률 상승 시그널
- 역전 해소(bear steepening) 시점이 실제 침체와 더 가깝게 연동
- 현재 레벨이 아닌 **추세 방향(기울기 변화)**이 더 중요

## 신용 시장

### 스프레드 수준별 해석
| 구간 | HY OAS | 해석 |
|------|--------|------|
| 타이트 | < 300bp | 리스크 선호 극대화, 주의 필요 |
| 정상 | 300–500bp | 균형 상태 |
| 확대 | 500–800bp | 신용 스트레스 |
| 위기 | > 800bp | 시스템 위기 (2008: 1900bp, 2020: 1100bp) |

### 크레딧 사이클 위치 판단
1. 스프레드 레벨 (현재 수준)
2. 스프레드 변화 속도 (급등 여부)
3. 발행 시장 활동 (신규 발행 급감 = 위험 신호)

## 크로스에셋 상관관계

### 정상 vs 위기 레짐
| 에셋 쌍 | 정상 | 위기 |
|---------|------|------|
| 주식-채권 | 음의 상관 (안전자산 효과) | 양의 상관 (동반 하락) |
| 주식-금 | 낮은 상관 | 음의 상관 (금 safe-haven) |
| 달러-EM | 음의 상관 | 강한 음의 상관 (달러 강세 = EM 압박) |

### 상관관계 붕괴 시그널
- 주식-채권 동반 상승/하락 → 레짐 전환 경고
- 달러-금 동반 강세 → 극단적 안전자산 선호

## 🔴 2026년 현재 시장 시그널 스냅샷

```
기준일: 2026-02-20

수익률 곡선:
  10Y-2Y = +65bp (역전 해소, 정상 기울기 회복)
  → 침체 선행 신호 소멸. 단, bear steepening 여부 모니터링 필요

신용 스프레드:
  HY OAS ≈ 300bp (5년 범위의 5퍼센타일 = 역사적 최저권)
  → 시장이 '완벽한 시나리오'를 가격에 반영 중
  → 상방 여력 제한, 하방 리스크 비대칭적으로 큰 구간

VIX:
  5년 범위의 12퍼센타일 = 극도의 낙관/컴플레이선시
  → 역발상적 위험 신호: 시장이 자기만족적일 때 충격에 취약

종합 판단:
  시장은 현재 near-perfect 시나리오를 pricing 중
  → 관세·Fed 의장 교체·지정학 중 하나라도 어긋나면 스프레드·VIX 급변 가능
  → 가격이 낮을 때 헤지 비용이 가장 저렴한 구간
```

**현재 구간 해석 규칙:**
- HY OAS < 300bp + VIX < 15 = 컴플레이선시 구간 → 헤지 검토
- 스프레드·VIX 동시 확대 시 속도 중요 (빠를수록 시스템 이슈)
- 수익률 곡선 bear steepening = 침체 현실화 경고 (역전 해소 이후 패턴)

## 이상 탐지 기준

### Z-Score 기반
- |Z| > 2.0: 주의 (이상 가능성)
- |Z| > 3.0: 이상 탐지
- |Z| > 4.0: 극단적 이상 (역사적 사건급)

### 복합 시그널 해석
이상 탐지 시 반드시 확인:
1. 동일 방향 다중 시그널 여부 (2개 이상 = 신뢰도 증가)
2. 펀더멘털 근거 존재 여부
3. 기술적 과매수/과매도 수준 (RSI, 볼린저)
4. 시장 구조적 요인 (대형 만기, 정책 일정)
