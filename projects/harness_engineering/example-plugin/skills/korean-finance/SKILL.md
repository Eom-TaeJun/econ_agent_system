## 한국-미국 연계 분석

### 금리 차이 (BOK - Fed)
- **마이너스(-) 구간**: 자본 유출 압력, 원화 약세 우려
- **한국은행 딜레마**: 물가 안정 vs 환율 방어 vs 성장 지지
- 현재 BOK는 Fed를 후행하는 경향 (평균 6-12개월 지연)

### 환율 드라이버 (원달러)
1. **미국 실질금리** (가장 강한 결정 요인)
2. **반도체 수출** (삼성/SK하이닉스 → 경상수지)
3. **중국 경기** (한국 수출의 약 25%)
4. **외국인 주식 순매수** (단기 수급)
5. **글로벌 리스크 온/오프** (안전자산 선호 시 원화 약세)

---

## 한국 경기사이클 특수성

- **반도체 수출 사이클**: IT 다운사이클 = 한국 성장 압력 핵심 변수
- **중국 의존도**: 중국 PMI 하락 → 한국 수출 선행 지표
- **가계부채**: GDP 대비 세계 최고 수준 → 금리 인상 민감도 극대화
- **부동산 연계**: 주택가격 → 가계소비 연동성 높음

## 🔴 2026년 한국 현황 업데이트

### BOK 기준금리 & 통화정책
```
기준금리: 2.50% (2026년 1월 기준, 5연속 동결)
누적 인하: 2024년 10월 이후 총 100bp 인하 후 동결
2026년 전망: 추가 인하 없이 2.50% 유지 전망 (DBS 등)

동결 이유:
  1. 원화 약세 (16년래 최저권 1,470 KRW/USD)
     → 추가 인하 시 자본 유출 위험
  2. 원화 약세 → 수입 물가 상승 → 인플레이션 재점화 위험
  3. 주택가격 상승 우려 (금융안정 우선)
```

### 원달러 환율 (KRW/USD)
```
현재: ~1,470 KRW/USD (16년래 최저권)

환율 드라이버 (2026):
  하락 요인 (원화 강세 방향):
    - Fed 금리 인하 → 달러 약세
    - BOK 인하 사이클 종료 → 금리차 축소
    - 미중 무역 긴장 완화 기대
  상승 요인 (원화 약세 방향):
    - 글로벌 관세 불확실성 → 리스크 오프
    - 한국 수출 둔화 (반도체 외 약세)
    - 정치 불확실성

목표치 전망: 1,375 (2026년 중반) → 1,400 (연말)
→ 원화 완만한 강세 복귀 기대
```

### 한국 거시 지표 (2026 전망)
```
GDP 성장률: 1.8% (BOK 전망, 완만한 회복)
CPI: 2.1% (헤드라인) / 2.0% (코어) → 목표 수준
2025 GDP 성장률: 1.0%로 하향 (수출 부진, 내수 약세)

주요 리스크:
  - 미국 관세 → 반도체 수출 영향
  - 중국 경기 회복 속도
  - 국내 정치 불확실성 지속
```

### 관세가 한국에 미치는 영향
```
한국 수출의 對美 비중: 약 18%
반도체: 전략물자로 일부 예외 적용 (하이닉스, 삼성 수혜)
자동차: 관세 직격 → 현대/기아 수출 마진 압박
철강: 추가 관세 위험
→ 업종별 관세 익스포저 반드시 구분해서 분석
```

## 리포트 언어 규칙

한국어 리포트 작성 시:
- 경제 전문 용어는 영어 병기: 수익률 곡선(Yield Curve), 기저효과(Base Effect)
- 수치는 한국식 단위: 억원, 조원, 원(₩)
- 시점 표현: "전월대비(MoM)", "전년대비(YoY)", "전분기대비(QoQ)"
- 금융감독원/한국은행 공식 용어 우선 사용
